Sobre

Sistema simples e rápido para formatar todo um texto.

Em cada lado do botão formatar existe uma seta, esta seta serve para voltar ou avançar, toda formatação feita é gravada em um histórico temporário, ou seja, todo o histórico será excluído se a página for fechada ou atualizada.

Adição

Com o método de formatação de texto, prefixo/sufixo, cada parágrafo ou item de lista será alterado conforme os parâmetros passados (prefixo e sufixo). Você pode usar esse método para adicionar um ponto no final de cada parágrafo por exemplo.

Usando o método novas quebras de linhas o formatador vai remover todas as quebras existentes e adicionar novas conforme a quantidade de quebras informadas no final de cada parágrafo ou item de lista.

Conversão

Método 1º letra para maiúscula pega a primeira letra de cada parágrafo ou item de uma lista e transforma em maiúscula.

O método tamanho da fonte converte todo o texto para maiúscula ou minúscula, você escolhe.

Remoção

Método quebras de linha vai remover as quebras de linha presentes no texto, você pode escolher entre remover todas ou apenas as quebras de linha duplicadas.

Espaços, vai remover todos os espaços ou apenas os espaços duplicados, você decide.

Outros

Indentar parágrafos vai indentar todos os parágrafos usando a quantidade de espaços informada.

Inverter texto vai inverter todo o texto, se você clicar novamente o texto voltará ao seu estado normal.

Último método, Localizar e substituir vai procurar o alvo e substituir pelo caractere informado ou vai simplesmente remover o alvo, caso não informe um substituto.
